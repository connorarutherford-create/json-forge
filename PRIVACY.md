# Privacy Policy — JSON Forge

**Last updated:** July 2026

JSON Forge does not collect, store, transmit, or share any personal data.

## Data Handling

| Data type | Handling |
|-----------|----------|
| **JSON you paste** | Processed entirely in your browser. Never sent to any server. |
| **Settings** | Stored locally in `chrome.storage.sync`. Not accessible to us. |
| **Usage stats** | Daily format count stored locally. Not transmitted. |
| **Pro license** | Chrome.storage flag stored locally. No license server. |

## Third Parties

JSON Forge makes no network requests. It loads no external resources, no analytics, no tracking scripts, no fonts from CDN. Everything runs from the extension's local files.

## Changes

If this policy changes, the extension version number will increment and you'll be notified via Chrome's auto-update mechanism.
